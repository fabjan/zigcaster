## Zigcaster

Trying to learn some Zig, using https://github.com/ssloy/tinyraycaster

## Requirements

* [zig] 0.9.1

[zig]: https://ziglang.org

## Building

```shell
./Taskfile build
```

## Running directly

```shell
./Taskfile run
```

## Testing

```shell
./Taskfile test
```
